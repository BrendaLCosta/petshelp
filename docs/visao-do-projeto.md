# 📋 Documento de Visão - PetsHelp

## 1. Introdução

### 1.1 Propósito
O PetsHelp é um sistema de gestão para ONGs e abrigos de animais.

### 1.2 Escopo
- Cadastro de doadores
- Registro de doações
- Controle de status
- Emissão de recibos
- Relatórios

## 2. Requisitos Funcionais

| ID | Requisito | Prioridade |
|----|-----------|------------|
| RF01 | Cadastrar doador | Essencial |
| RF02 | Listar doadores | Essencial |
| RF03 | Registrar doação | Essencial |
| RF04 | Atualizar status da doação | Essencial |
| RF05 | Emitir recibo | Importante |

## 3. Tecnologias

| Camada | Tecnologia |
|--------|------------|
| Frontend | React |
| Backend | Node.js + Express |
| Banco | MySQL |
| ORM | Sequelize |
